# Cindy Care 🐱



แอปผู้ช่วยโภชนาการสำหรับแมว "Cindy" — วางแผนมื้ออาหาร, คะแนนโภชนาการ, สต็อก/รายการซื้อ, สุขภาพ (น้ำหนัก/ขับถ่าย/ปัสสาวะ/น้ำดื่ม/ยา/หาหมอ) และปฏิทินรายวัน



เว็บ **static ล้วน** (HTML + CSS + Vanilla JS) ไม่มี build step เก็บข้อมูลในเครื่องด้วย `localStorage`



## โครงสร้างไฟล์

```

cindy/

├─ index.html      # โครงหน้า

├─ style.css       # ธีม (Light/Dark)

├─ script.js       # ลอจิกทั้งหมด

├─ assets/         # (ไม่บังคับ) รูปสินค้า ตั้งชื่อตาม id เช่น assets/hills_cd.png

└─ README.md

```



## รันในเครื่อง

ดับเบิลคลิก `index.html` เปิดในเบราว์เซอร์ได้เลย



## ขึ้น GitHub Pages (ฟรี)

1. สร้าง repository ใหม่ แล้วอัปโหลดไฟล์ทั้งหมด (index.html, style.css, script.js + โฟลเดอร์ assets ถ้ามี) ให้ `index.html` อยู่ที่ราก

2. ไปที่ **Settings → Pages**

3. Source: เลือก **Deploy from a branch** → branch `main` → folder `/ (root)` → Save

4. รอสักครู่ จะได้ลิงก์ `https://<username>.github.io/<repo>/`



> ถ้า index.html ไม่ได้อยู่ราก (อยู่ในโฟลเดอร์ย่อย) ให้เลือก folder เป็นโฟลเดอร์นั้น หรือย้ายไฟล์ออกมาที่ราก



## รูปสินค้า

- ใส่รูปจริงได้ 2 วิธี: อัปโหลดในแอป (ปุ่มแก้ไขอาหาร) หรือวางไฟล์ในโฟลเดอร์ `assets/` ตั้งชื่อตาม id ของอาหาร เช่น `assets/peace.png`

- ถ้าไม่มีรูป จะแสดง placeholder อักษรย่อให้อัตโนมัติ



## หมายเหตุ

- ใช้อินเทอร์เน็ตเฉพาะ: Google Fonts และ Tesseract.js (เฉพาะตอนกด "อ่านฉลากจากรูป" OCR) — ส่วนอื่นทำงานออฟไลน์ได้

- ข้อมูลเก็บในเบราว์เซอร์ของแต่ละเครื่อง (ไม่ซิงก์ข้ามอุปกรณ์) — สำรอง/กู้คืนได้ที่ ⚙️ ตั้งค่า → สำรองข้อมูล (.json)

- มี cache-busting `?v=` ที่ลิงก์ไฟล์ ถ้าแก้โค้ดแล้วไม่อัปเดต ให้เพิ่มเลขเวอร์ชัน
